# C_sharp_Learning

A small repository for learning C# and preparing for the final 42 Common Core project.

## Overview

This repo contains practice code, exercises, and small projects to build familiarity with C# language features, the .NET runtime, and common development workflows.

## Prerequisites

- Install .NET SDK (6.0 or later). See https://dotnet.microsoft.com/

## Getting Started

1. Clone the repository:

	git clone https://github.com/medob6/C_sharp_Learning.git

2. Navigate into a project folder and build with `dotnet`:

	dotnet build

3. Run the project (if it has an entry point):

	dotnet run --project <ProjectFolder>

## Repository Structure

- (root) — general notes and learning resources
- /Exercises — small example projects and code snippets
- /Projects — larger example projects (if added)

Add folders as you create practice projects.
